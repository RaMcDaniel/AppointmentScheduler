Appointment Scheduler is a program designed to create and manage appointments for various users and customers for an
organization. You can look through appointments, edit them, and run related reports.

Directions to run the program:
1. Launch the application
2. login using one of the following username and password combinations:
    test:test; admin:admin
3. Navigate between customer information, appointment information, and the 3 reports via the 5 buttons on the welcome screen.


Extra report description:
    The extra report can be accessed via the button labeled "reports by customer" on the welcome screen. This report
allows the user to choose a customer, and then see a table of all appointments filtered by that customer. This is useful
to determine if there are any appointments associated with a particular customer before you delete them from the system.
